# Odin

*Compliance management platform and visual hub for the Odin ecosystem.*

Odin is the web-based interface for facility management, inspections,
regulatory tracking, and — when other ecosystem tools are installed — a
unified dashboard for knowledge management and document generation.

## Ecosystem

Odin is part of the [Asgard](https://github.com/pharomwinters/asgard) ecosystem:

| Component | Role |
|-----------|------|
| **Odin** | Web UI + compliance platform |
| **[Muninn](https://github.com/pharomwinters/muninn)** | Knowledge base + semantic search |
| **[Huginn](https://github.com/pharomwinters/huginn)** | Markdown-to-PDF document generation |
| **[Bifrost](https://github.com/pharomwinters/bifrost)** | Infrastructure daemon + Heimdall config manager |

## Status

Early development. Architecture design in progress.
